Вопрос:

	Что выведет данная программа и почему?

func update(p *int) {
  b := 2
  p = &b
}

func main() {
  var (
     a = 1
     p = &a
  )
  fmt.Println(*p)
  update(p)
  fmt.Println(*p)
}

Ответ:

    1 // Потому что p в этот момент указывает на a, которое равно 1
    1 // Потому что p в этот момент указывает на a, которое равно 1,
    // В GO аргументы всегда передаются по значению, а не по ссылке. В update
    // просто был изменен указатель, который является локальной переменной
